Схема интеграции с 1С
HTTP-сервис через расширение конфигурации 1С:Розница KZ
01
GET-запрос с сайта
При загрузке страницы JavaScript отправляет fetch() к HTTP-сервису 1С для получения номенклатуры, цен и остатков
02
Расширение конфигурации
Расширение (без изменения типовой конфигурации) добавляет HTTP-сервис, формирующий JSON из справочника «Номенклатура»
03
JSON → Каталог
Сайт получает массив товаров, рендерит карточки с актуальными ценами и остатками склада без перезагрузки
04
Заказ → 1С
POST-запрос с данными корзины создаёт «Заказ клиента» в 1С:Розница KZ и резервирует товар на складе
